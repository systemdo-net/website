## 📖 About

This is the official website of **SystemDo**, a professional enterprise software development company. Built with **Nuxt 3**, the site is optimized for both traditional search engines (SEO) and AI-powered search (GEO — Generative Engine Optimization).

The website showcases SystemDo's full range of enterprise software services, case studies, and technical capabilities.

## ✨ Features

- ⚡ **Nuxt 3** with SSR for fast initial loads and excellent SEO
- 🎨 Responsive design with dark/light mode support
- 🔍 SEO-optimized with structured data and meta tags
- 🤖 GEO-friendly content structure for AI search engines (ChatGPT, Gemini, Claude, Perplexity)
- 📱 Mobile-first responsive layout
- 🌍 Multi-language ready architecture
- 📊 Integrated analytics (GA4, Meta Pixel)

## 🛠️ Tech Stack

| Category | Technology |

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
website/

├── nuxt.config.ts  # Nuxt configuration
```
