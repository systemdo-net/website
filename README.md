## 📖 项目介绍

SystemDo 企业官网，基于 **Nuxt 3** 构建的高性能 SSR 网站。面向海外华人企业客户，展示公司品牌形象、六大核心服务、项目案例与技术实力。

网站同时针对传统搜索引擎（SEO）和 AI 搜索引擎（GEO — 生成式引擎优化）进行了深度优化，确保在 Google、ChatGPT、Gemini、Claude、Perplexity 等平台上都能获得良好的品牌曝光。

## ✨ 核心特性

- ⚡ **Nuxt 3 SSR** — 服务端渲染，首屏加载快，搜索引擎友好
- 🎨 **响应式设计** — 支持明暗主题，PC/平板/手机全适配
- 🔍 **SEO 深度优化** — 结构化数据、Meta 标签、Sitemap 自动生成
- 🤖 **GEO 友好架构** — 内容结构清晰，便于 AI 搜索引擎理解和引用
- 📱 **移动优先** — 针对海外华人常用的移动端场景优化
- 🌍 **多语言架构** — 支持未来扩展多语言版本
- 📊 **数据驱动** — 集成 GA4、Meta Pixel 等分析工具

## 🚀 本地开发

```bash
npm install
npm run dev
npm run build
npm run preview
```

```
website/

├── pages/          # 路由页面

└── package.json    # 依赖管理
```
