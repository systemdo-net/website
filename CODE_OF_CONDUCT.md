# 行为准则

## 我们的承诺

作为 **SystemDo** 社区的成员、贡献者和领导者，我们承诺让每个人都能在友好、尊重、包容的环境中参与——不论其年龄、体型、残障、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向。

我们承诺以促进开放、欢迎、多元、包容和健康社区的方式行动和互动。

## 行为标准

### 积极行为

- 对他人展现同理心和善意
- 尊重不同的意见、观点和经历
- 给予并优雅地接受建设性反馈
- 为自己的失误承担责任并向受影响者道歉
- 关注对整体社区最有利的事情

### 不可接受的行为

- 使用性化语言或图像，以及任何形式的性骚扰
- 恶意挑衅、侮辱或贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可公布他人隐私信息
- 其他在专业环境中可合理视为不当的行为
